# LockFlow
LockFlow is a toolkit that combines two tools in one platform, designed to enhance your security. Efficient, easy to use.
 ![Image Alt](https://github.com/SonicExE404/LockFlow/blob/main/LockFlow.png)

# Tool [1]: EzCrypt
A tool that can help you create secure password , that you can remember with ease.
  ![Image Alt](https://github.com/SonicExE404/LockFlow/blob/main/EzCrypt.png)

# Tool [2]: Pass-Meter
Just a simple tool that can tell you the strength of your password.
  ![Image Alt](https://github.com/SonicExE404/LockFlow/blob/main/Pass-Meter.png)
